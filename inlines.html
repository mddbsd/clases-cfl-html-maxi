El elemento span

El elemento span, es un contenedor del tipo inline, utilizado para marcar parte de un texto o parte de un documento
Este elemento, al igual que el div se suele utilizar con los atributos style, class, e id
Estos elementos en conjunto con css se pueden utilizar para darle formato a partes del texto0
